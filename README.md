# Strata

`Strata` is a simple, lightweight CSV file editor written in Rust.

# Keybindings

## Table Select Mode

| Keybinding | Description |
|------------|-------------|
| a | Add new table |
| o | Open CSV file |
| j, ↓ | Move down |
| k, ↑ | Move up |
| s | Save |
| q | Quit |

## Table View Mode

| Keybinding | Description |
|------------|-------------|
| h, ← | Move left |
| j, ↓ | Move down |
| k, ↑ | Move up |
| l, → | Move right |
| a, e, Enter | Edit cell |
| r | Add new row |
| R | Delete row |
| c | Add new column |
| C | Delete column |
| s | Save |
| q, Esc | Quit Table View Mode|
